# React Practice Projects 🚀

This repository contains multiple React practice projects created while learning React fundamentals, routing, component communication, layouts, and database integration.

---

## 📂 Projects Included

### 📁 RJS_Outlet_layout
A React project demonstrating the use of **React Router Outlet** and **Layout Components** for creating reusable page layouts.

**Topics Covered**
- React Router DOM
- Outlet
- Layout Components
- Nested Routing
- Navigation

---

### 📁 React_DB
A React CRUD application connected with a backend database.

**Topics Covered**
- CRUD Operations
- API Integration
- Axios
- Forms
- State Management
- MongoDB
- Express.js
- Node.js

---

### 📁 React_Project
A collection of React practice programs covering the core concepts of React.

**Topics Covered**
- Components
- JSX
- Props
- State
- Event Handling
- Conditional Rendering
- Lists
- Forms
- Hooks

---

### 📁 React_Props
A beginner-friendly project focused on understanding React Props and component communication.

**Topics Covered**
- Functional Components
- Props
- Parent to Child Communication
- Dynamic Data
- Reusable Components

---

### 📁 react_router

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/your-repository.git
```

Go to the desired project folder:

```bash
cd React_Project
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm start
```
